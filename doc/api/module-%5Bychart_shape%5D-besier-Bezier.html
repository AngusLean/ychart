Class: Bezier

[ychart/shape]-besier~Bezier

new Bezier(beginpt, endpt, compact, systole)

绘制一条曲线路径. 调用此方法后当前的context将包含一条路径.

Parameters:
Name Type Description
beginpt Array.number

x,y 开始点坐标
endpt Array.number

x,y 结束点坐标
compact number

0-10 表示该波浪线的紧凑程度,即每个弯曲的距离比例
systole number

0-4 波浪的起伏程度, 表现为上下波动程度. 1为标准.小于1就是缩放.1-4就是放大
